# Parcel Boilerplate

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run start
```

### Compiles and minifies for production
```
npm run build-prod
```

### Compiles and minifies for production without source mapping
```
npm run build-prod-no-map
```

### Clean
```
npm run clean
```
